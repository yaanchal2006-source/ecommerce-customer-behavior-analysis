# E-Commerce Customer Behavior Analysis Dashboard

## Project Overview
This project is built using Power BI to analyze customer behavior in an e-commerce platform.

## Tools Used
- Power BI
- Excel

## Files
- ecommerce project.pbix (Main Dashboard)
- Dataset files

## Key Features
- Total Visitors Analysis
- User Activity Tracking
- Top Viewed Products
- Conversion Funnel
- User Activity Over Time

## Objective
To understand customer behavior and improve business decisions using data visualization.
